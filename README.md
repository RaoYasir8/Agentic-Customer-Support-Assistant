# Agentic AI Customer Support Assistant

This project is a production-style AI customer support assistant built with FastAPI, LangChain agents, custom tools, conversation memory, and Groq LLM.

## Features

- AI-powered customer support assistant
- LangChain agent with tool calling
- Custom tools for order status, policies, product info, and ticket creation
- Conversation memory for multi-turn interactions
- FastAPI backend
- HTML/CSS/JavaScript frontend
- Groq LLM integration
- REST API endpoint
- Docker support

## Tech Stack

- Python
- FastAPI
- LangChain
- LangChain Groq
- Groq LLM API
- HTML
- CSS
- JavaScript
